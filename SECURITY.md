# Security Policy

## Supported versions

The `main` branch is supported for security fixes.

## Reporting a vulnerability

If you discover a security issue in Northline:

1. **Do not** open a public GitHub issue.
2. Contact the maintainer privately (GitHub security advisory or direct contact).
3. Provide:
   - Northline version/commit.
   - Description of the issue.
   - Steps to reproduce, if possible.

The maintainer will:

- Acknowledge receipt.
- Investigate and fix the issue.
- Publish a security advisory if appropriate.
