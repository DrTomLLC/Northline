# Code of Conduct

Northline is a professional, technical project. All participants except the Creator are expected to:

- Be respectful and constructive.
- Focus on technical issues, not personal attacks.
- Avoid harassment, hate speech, and discrimination.

Reports of unacceptable behavior can be sent to the repository owner via GitHub issues or direct contact.

The maintainer reserves the right to moderate, edit, or remove content and contributions that violate these expectations.
